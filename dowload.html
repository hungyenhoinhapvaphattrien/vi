HƯỚNG DẪN CÀI ĐẶT ỨNG DỤNG

Tải về và cài đặt ứng dụng Hưng Yên Mới trên điện thoại Android để trải nghiệm tốt hơn

Hưng Yên Mới
Phiên bản 1.0
Kích thước: 15MB
TẢI ỨNG DỤNG NGAY

Yêu cầu hệ thống

  • Android 5.0 trở lên
  • RAM 1GB trở lên
  • Dung lượng trống 20MB

Lưu ý quan trọng

Ứng dụng này không có trên Google Play Store, bạn cần cài đặt thủ công (sideload). Xin vui lòng làm theo hướng dẫn bên cạnh.

1

Tải file APK

Bấm "Tải ứng dụng ngay và tiến hành tải file APK về máy.

2

Mở file APK đã tải

Vào trình quản lý file, tìm đến thư mục Downloads và nhấn vào file APK vừa tải để bắt đầu cài đặt.

3

Hoàn tất cài đặt

Nhấn "Cài đặt" và chờ quá trình hoàn tất. Sau khi cài đặt xong, bạn có thể mở ứng dụng ngay hoặc nhấn "Mở" từ màn hình cài đặt.
